# jsbrowser

jsbrowser - браузер, работающий на веб-странице, как обычный браузер. Поддерживает вкладки и асинхронную работу). Примеры для вдохновения - lynx, links, w3m

## Участие

* Обсуждение в канале #hexlet-volunteers слака
* Стажировка Хекслета

## Технологии

* React, Redux
* Boostrap
